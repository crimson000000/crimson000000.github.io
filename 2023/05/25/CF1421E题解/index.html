本题作为一道本人思考了50分钟没想出来的大思维题，我觉得可以用来扩宽一下大家的视野。

本题中，我们每次都会选取两个相邻的数 aia_iai​ 和 ai+1a_{i+1}ai+1​，同时将这两位变为一位，这一位上填的数为 −(ai+ai+1)-(a_i+a_{i+1})−(ai​+ai+1​)。

很容易想到的一个 O(n3)O(n^3)O(n3) 的 dp 做法是区间 dp，设 f[l,r]f[l,r]f[l,r] 为区间 [l,r][l, r][l,r] 的最大值，g[l,r]g[l,r]g[l,r] 为区间 [l,r][l, r][l,r] 的最小值，我们就可以得到如下方程：

f[l,r]=f[l,r]=max⁡k=lr−1{−(g[l,k]+g[k+1,r])}f[l, r] = f[l,r]=\max \limits _{k=l}^{r-1}\{ -(g[l, k]+g[k+1,r]) \} f[l,r]=f[l,r]=k=lmaxr−1​{−(g[l,k]+g[k+1,r])}

g[l,r]=g[l,r]=min⁡k=lr−1{−(f[l,k]+f[k+1,r])}g[l, r] = g[l,r]=\min \limits _{k=l}^{r-1}\{ -(f[l, k]+f[k+1,r]) \} g[l,r]=g[l,r]=k=lminr−1​{−(f[l,k]+f[k+1,r])}

但是题目数据范围太大，无法承受。

我们可以考虑一下答案有没有什么性质。

一个比较显然的性质是：答案一定是 aaa 序列所有数加加减减得到的，更严谨地说，aaa 序列中每个数对答案的贡献取决于它在答案中的正负号。

因此我们可以考虑什么样的正负号排列才是合法的。

我们可以枚举一下 n=1,2,3n = 1, 2, 3n=1,2,3 的情况。

n=1n = 1n=1

符合要求的序列有 +。

n=2n=2n=2

符合要求的序列有 --。

n=3n=3n=3

符合要求的序列有 ++-, -++

能不能发现什么性质

我们统计一下 - 的数量 ppp, + 的数量 qqq，我们可以很难发现一个规律：2p+q≡1(mod3)2p+q \equiv 1 \pmod{3}2p+q≡1(mod3)

证明：显然对于 n=1,2,3n=1,2,3n=1,2,3 的情况都成立，考虑当长度为 nnn 时的情况：

每个情况一定都是两种合法情况的组合，那么：

−[(2p1+q1)+(2p2+q2)]≡−(1+1)(mod3)≡1(mod3)\begin{aligned} -[(2p_1+q_1)+(2p_2+q_2)] &\equiv -(1+1) \pmod{3} \\ &\equiv 1 \pmod{3} \end{aligned} −[(2p1​+q1​)+(2p2​+q2​)]​≡−(1+1)(mod3)≡1(mod3)​

证毕

但是还有一种情况：+-+-+-+-......。这种情况是不合法的。所以我们接下来还要考虑上这点。

想到这个性质之后我们就可以开始 dp 了，我们设 f[i][j][k][u]f[i][j][k][u]f[i][j][k][u] 表示当前考虑到第 iii 个数，2p+q2p+q2p+q 在模 333 意义下为 jjj，上一个符号为 kkk，是否出现了两个相同符号相邻 uuu。

状态设计出来后就可以开始 dp 了。

f[i+1][(j+2) mod 3][0][u∣(k==0)]←f[i][j][k][u]−ai+1f[i+1][(j+2)\bmod 3][0][u|(k==0)]\leftarrow f[i][j][k][u]-a_{i+1} f[i+1][(j+2)mod3][0][u∣(k==0)]←f[i][j][k][u]−ai+1​

f[i+1][(j+1) mod 3][1][u∣(k==1)]←f[i][j][k][u]+ai+1f[i+1][(j+1)\bmod 3][1][u|(k==1)]\leftarrow f[i][j][k][u]+a_{i+1} f[i+1][(j+1)mod3][1][u∣(k==1)]←f[i][j][k][u]+ai+1​

然后就可以愉快 AC 力！
